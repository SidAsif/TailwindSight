Privacy Policy

Effective Date: November 9, 2025

TailwindSight ("the Extension") is a Chrome Developer Tool that allows users to inspect and modify Tailwind CSS classes on webpages in real time. Your privacy is important to us, and this policy explains how we handle data when you use the Extension.

1. Data Collection

TailwindSight does not collect, transmit, or store any personal information, browsing data, or usage analytics. All operations take place entirely within your browser, and no information leaves your local environment.

2. Data Access and Usage

When you activate TailwindSight on a webpage, the Extension temporarily reads HTML element class attributes for the sole purpose of displaying and editing Tailwind CSS classes in the visual inspector. This data is processed locally and temporarilyâ€”it is not saved, transmitted, or shared.

3. Permissions

TailwindSight requests minimal Chrome permissions necessary to function:

  • activeTab: Grants temporary access to the currently active tab when you click the extension icon and activate inspection mode.
  • storage: Saves the inspection mode state (on/off) locally in your browser. Data never leaves your browser.
  • host permissions (<all_urls>): Required to inspect and modify Tailwind CSS classes on any webpage you choose to work on.

These permissions are used strictly to support the Extension's single purpose: enabling Tailwind CSS class inspection and editing in real time.

4. Remote Resources

TailwindSight uses the Inter font from Google Fonts to ensure consistent typography. This font is loaded via standard CSS and does not execute any JavaScript or collect data.

5. Third-Party Services

TailwindSight does not integrate with or send information to any third-party analytics, advertising, or tracking services.

6. Data Sharing

TailwindSight does not share, sell, or disclose any data to third parties. No data is ever transmitted outside the user's local browser.

7. Security

Since all processing occurs locally in the user's browser, no external servers are involved. The Extension is Manifest V3 compliant and follows Chrome's latest security best practices.

8. Changes to This Policy

We may update this privacy policy from time to time to reflect product improvements or changes in legal requirements. The "Effective Date" at the top of this page will be updated accordingly. Continued use of TailwindSight after any changes constitutes acceptance of the updated policy.
